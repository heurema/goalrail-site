# Polly

Polly - многоагентный coding orchestrator. Он принимает задачу, выделяет
подзадачи, запускает sub-agents и собирает результат через review.

```
goalrail polly
goalrail          # тоже запускает Polly как default workflow
```

## Когда использовать

- Большая coding task делится на независимые части.
- Нужен cross-review между агентами или моделями.
- Нужно сравнить несколько подходов без ручной координации worktrees.

## Как устроено

Polly использует skills вроде `/fanout`, `/cross-review` и `/investigate`.
Каждый sub-agent получает свой контекст и может работать в отдельном worktree.
В UI вы видите orchestrator, sub-agents, shells, todos и созданные файлы.
