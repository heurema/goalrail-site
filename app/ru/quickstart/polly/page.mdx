# Туториал: многоагентная coding orchestration

Polly - встроенный orchestrator для coding tasks. Он разбивает задачу на
подзадачи, запускает несколько AI-агентов параллельно и добавляет review между
ними. Этот quickstart показывает, как это выглядит в UI.

**Время:** около 5 минут. **Все запускается локально.**

## 1. Запустите Polly

```
goalrail polly
```

Команда откроет web UI на `http://localhost:6767`. Если выполнить просто
`goalrail`, Polly тоже стартует как default workflow.

## 2. Дайте задачу

```
Refactor the authentication module into separate files for OAuth, JWT, and session handling. Add tests for each.
```

Polly разложит задачу на части, запустит sub-agents и будет координировать
результат. Один агент может писать код, другой - тесты, третий - review.

Правая панель содержит вкладки **Files**, **Agents**, **Shells** и **Todos**.
Во вкладке **Agents** видно каждого sub-agent и его статус.

- Откройте sub-agent, чтобы посмотреть его диалог, файлы и terminal output.
- Перейдите в **Shells**, чтобы увидеть запущенные процессы.
- Напишите Polly или конкретному sub-agent, если хотите изменить ход работы.
